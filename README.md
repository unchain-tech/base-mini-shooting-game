# unchain-base-mini-app

BaseのMiniAppKitを使ってシューティングゲームアプリを作るリポジトリ

## シューティングゲームのデモ

[デモ動画](https://x.com/haruki_web3/status/1962851527074472211)

## テンプレプロジェクト生成コマンド

```bash
npx create-onchain@latest --mini
```

## ビルド＆ローカルでの起動

- ビルド

  ```bash
  pnpm run build
  ```

- ローカルで起動

  ```bash
  pnpm run dev
  ```

## vercelにデプロイする

- プレビュー

    ```bash
    vercel
    ```

- リリース

    ```bash
    vercel --prod
    ```
